UpliftUnity

UpliftUnity is a dedicated platform aimed at promoting understanding, empathy, and unity. It allows users to discover stories, share their challenges, and connect with a supportive community. Cineverse

Cineverse is an interactive web application designed for movie enthusiasts. Users can view, search, add, edit, and delete movies. The application interfaces with a local server for movie data and leverages the OMDB API for comprehensive movie details. A standout feature is the real-time update of movie data, achieved through AJAX, offering a seamless user experience.

My role involved designing and building both front and back-end components.

Tools/Technologies: HTML5, CSS3, Bootstrap, JavaScript (ES6+), jQuery, AJAX, Local server, OMDB API, TMDB API

Weatherwise

Weatherwise offers real-time weather insights, including current conditions and a five-day forecast for any global location. Leveraging OpenWeatherMap and Mapbox APIs, users can interactively choose a location via a map or use the search functionality for dynamic weather updates. The application ensures precision with JavaScript's geolocation and Mapbox's geocoding, presenting a user-friendly and responsive interface.

My role involved designing and building both front and back-end components.

Tools/Technologies: HTML5, CSS3, Bootstrap, JavaScript (ES6+), jQuery, AJAX, OpenWeatherMap API, Mapbox API
